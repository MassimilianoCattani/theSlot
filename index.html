Le Grand Q

  • Aim of the game: matching user numbers with computer numbers within the same array position.
  • Budget works as a bank. Insert your initial budget and press OK (min = "1" max = "999").
  • Single bet is a withdrawal from budget. It can never be higher than current budget.
  • Push Grand Q button and good luck!
  • If budget is equal to "zero" -> GAME OVER
  • From drop-down menu Wanna play select END/RELOAD to leave/restart. You can quit at any time.
  • PRIZE:

    5/5: bet x 100 "Le Grand Q"

    4/5: bet x 20

    3/5: bet x 5

    2/5: bet x 1

    1/5: bet x 0

    0/5: bet x 0
